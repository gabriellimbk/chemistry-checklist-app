# Deducing oxidation number from formula

1. Deduce the oxidation number of nitrogen in the nitrite ion, NO₂⁻
Answer: Let the oxidation number of N be x.
  x + 2(-2) = -1
  x = +3

2. Deduce the oxidation number of oxygen in sodium peroxide, Na₂O₂
Answer: Let the oxidation number of O be x.
  2(+1) + 2(x) = 0
  x = -1

3. Deduce the oxidation number of sulfur in the thiosulfate ion, S₂O₃²⁻
Answer: Let the average oxidation number of S be x.
  2(x) + 3(-2) = -2
  x = +2
