# id-id

1. How do instantaneous dipole-induced dipole interactions arise?
Answer: Electrons are constantly moving. At any given moment, electron density in a molecule can be unsymmetrical, resulting in an instantaneous dipole which can then induce a dipole on a neighbouring molecule.

2. How does molecular size affect id-id interactions?
Answer: Larger molecules have larger and more polarisable electron clouds, hence have stronger id-id interactions.
